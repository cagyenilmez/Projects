# Optimization Projects

This repository contains small optimization and operations research projects built as Jupyter notebooks.

## Projects

- `Group17_Broccoli_Project.ipynb` - Final submitted coverage optimization LP problem for Broccoli Point location planning.
- `Nearest_Neighbour_VRP_Project.ipynb` - A simple capacitated vehicle routing problem solved with a nearest-neighbour construction heuristic and route visualization.

## Notes

The notebooks include saved outputs so the main results and visualizations can be viewed directly on GitHub.
